O que é VOCALOID?

Vocaloid é um software de síntese de voz desenvolvido
pela Yamaha, que permite criar canções com vozes geradas
por computador a partir de textos. Usuários podem compor
músicas utilizando diferentes vozes, conhecidas como
"vocaloids", que são representadas por personagens virtuais.
O software é amplamente utilizado na música pop e na cultura otaku, especialmente no Japão.
